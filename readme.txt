Welcome to Lohini project
==========================
The aim of this project is integration of interesting technologies and options
to Nette Framework.
Now heavily based on Kdyby (www.kdyby.org)

Requirements
------------
- PHP 5.3.2 or later
- Nette Framework - http://nette.org
